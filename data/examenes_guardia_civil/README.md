# Examenes Oficiales — Guardia Civil (Cabos y Guardias)

## Formato
- 100 preguntas + 5 reserva
- 4 opciones (A, B, C, D)
- Penalizacion: 1/3
- Tiempo: 60 minutos

## Instrucciones
1. Descargar PDF del examen y plantilla de respuestas
2. Guardar como `YYYY/examen.pdf` y `YYYY/plantilla.pdf`
3. Ejecutar: `pnpm parse:examenes --dir examenes_guardia_civil YYYY`
4. Ingestar: `pnpm ingest:examenes --oposicion guardia-civil`
